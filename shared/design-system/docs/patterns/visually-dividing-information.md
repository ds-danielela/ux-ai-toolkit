# Visually dividing information

**URL:** https://zeroheight.com/0b922d40b/v/latest/p/01ecef-visually-dividing-information

**Introduction:** Effective visual separation of information is a fundamental aspect of design that enhances clarity, readability, and user experience.

---

## Usage

In any interface, different types of content often coexist, and distinguishing these elements appropriately is crucial to guide the user's attention and understanding.

To visually separate information, choosing the right method depends on the specific use case and the importance of the separation.

### Spacing

Spacing is ideal when subtle separation is needed, or when elements are part of the same logical group but require some breathing room.

Guidelines:

* **Consistency:** Ensure consistent spacing across similar elements to maintain a cohesive design. Consult the [guidelines for spacing](https://zeroheight.com/0b922d40b/p/944aaa) for examples on which spacing to choose.
* **Responsive design:** Adjust the amount of spacing based on the screen size; larger screens can afford more whitespace (also see [responsive spacing and typography](https://zeroheight.com/0b922d40b/p/24f1b2)).

Example: Layout spacing within a page layout to visually separate the page header from the content area.

### Interpunct (·)

An interpunct (middle dot) can be used to visually divide textual information of the same hierarchy in a space-efficient and subtle way. An interpunct is i.e. suitable for inline separation of items within a single line of text, where items are related but need clear distinction.

Guidelines:

* **Minimalism:** Use the interpunct sparingly to avoid overwhelming the user with too much information. If more than three or four items need to be separated, consider using a different layout or structure.
* **Spacing:** Ensure there's equal spacing on either side of the interpunct to maintain balance and readability. Typically, a single space on either side of the interpunct works well.
* **Consistency:** Use the interpunct consistently for similar types of inline content. Ensure that the color and font size of the interpunct match the surrounding text.

Be cautious of how the interpunct behaves at the end of a line. Ensure that it does not end up at the start of a new line, which can disrupt readability.

Example: An interpunct can separate meta data within a card. Further possible usages: [container](https://zeroheight.com/0b922d40b/p/24be69), [list](https://zeroheight.com/0b922d40b/p/37109e), [media tile](https://zeroheight.com/0b922d40b/p/35ddbc), [table](https://zeroheight.com/0b922d40b/p/64dd20).

### Divider

Dividers are appropriate when a more explicit separation is needed between different sections, especially when those sections are distinct in content or functionality.

Guidelines:

* **Orientation:** Horizontal dividers are used to separate content vertically (e.g., between paragraphs or sections), while vertical dividers can be used in toolbars or between columns.
* **Spacing:** Add some spacing around the divider to prevent it from feeling cramped.

Examples: A vertical divider can be used to separate the header attributes within a second level page. Further possible usages: Focus mode toolbar, separation of items/rows in [accordion](https://zeroheight.com/0b922d40b/p/435154), [list](https://zeroheight.com/0b922d40b/p/37109e) and [table](https://zeroheight.com/0b922d40b/p/64dd20).

### Choosing the right method

When deciding which method to use for visual separation, consider the following factors:

* **Content hierarchy:** Determine the relationship between the elements. Use spacing for subtle grouping, dividers for distinct sections, and interpuncts for closely related inline items.
* **Visual emphasis:** Consider the importance of the separation. Whitespace offers the least visual emphasis, followed by interpuncts, with dividers providing the most distinct separation.
* **User experience:** Think about how the separation method will impact the user's ability to scan and understand the content. The goal is to enhance clarity and reduce cognitive load.

By applying these methods thoughtfully and consistently, you can create a clear, organized interface that improves user navigation and comprehension.

### Dos and don'ts

When using an interpunct, ensure that the color and font size match the surrounding text.
