# Contextual information

**URL:** https://zeroheight.com/0b922d40b/v/latest/p/2748b0-contextual-information

**Introduction:** Contextual information helps users understand form fields, actions, and interface elements by providing guidance at the right moment.

---

## Usage

Choose the appropriate pattern based on complexity and user needs. Common contextual information patterns include:

**Help text**

Use help text to provide concise guidance placed below a field. Common examples include i.e. password requirements.

**Hint text / placeholder**

Use hints inside a form field to suggest expected input, like, i.e. the date format.

![Contextual info: Hint text](https://zeroheight-uploads.s3.eu-west-1.amazonaws.com/88535669e9bfdad2f3881c?X-Amz-Algorithm=AWS4-HMAC-SHA256&X-Amz-Credential=AKIA3AVNYHQK6SJI2PWU%2F20260508%2Feu-west-1%2Fs3%2Faws4_request&X-Amz-Date=20260508T133044Z&X-Amz-Expires=604800&X-Amz-SignedHeaders=host&X-Amz-Signature=15a7f2b2c1be4b77be44f3536b8a441088290cbf17d525f3c0a99c0ce96d6aaa)

**Tooltip**

Use a tooltip:

* To describe icon buttons.
* To provide the user with relevant information that does not need to be seen at first glance.
* If text is truncated due to limited available space.

![Contextual info: Tooltip](https://zeroheight-uploads.s3.eu-west-1.amazonaws.com/0f65ef1004bfc53a69cf97?X-Amz-Algorithm=AWS4-HMAC-SHA256&X-Amz-Credential=AKIA3AVNYHQK6SJI2PWU%2F20260508%2Feu-west-1%2Fs3%2Faws4_request&X-Amz-Date=20260508T133044Z&X-Amz-Expires=604800&X-Amz-SignedHeaders=host&X-Amz-Signature=4ebcc0a3cd1897accb74975c534cc754082847f904e0b6ff6f0a31653c46c535)

**Info modal**

Use an info modal for detailed explanations, including images or complex information. In forms, the info modal is primarily triggered by an info button.

![Contextual info: Info modal](https://zeroheight-uploads.s3.eu-west-1.amazonaws.com/c3ff81912ab08b6c5e4025?X-Amz-Algorithm=AWS4-HMAC-SHA256&X-Amz-Credential=AKIA3AVNYHQK6SJI2PWU%2F20260508%2Feu-west-1%2Fs3%2Faws4_request&X-Amz-Date=20260508T133044Z&X-Amz-Expires=604800&X-Amz-SignedHeaders=host&X-Amz-Signature=0d98fa90fea79f44a7eb95fd61d7acd9678af500ae5ca8baa68a67a3c85edc75)

In other areas, the info modal could also be triggered by a link or button.
