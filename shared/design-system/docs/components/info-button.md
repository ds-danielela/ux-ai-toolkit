### Size and scaling

The info button has a fixed size of 24px * 24px.

---

## Anatomy

The info button contains the "Info-circle" icon.

---

## Usage

Use info buttons to display more information on demand, i.e. in a form. The info button can be used to trigger an info modal.

### Do's and don'ts

Only use the info button in combination with label, i.e. in a form. Do not combine it with other buttons in a toolbar.
