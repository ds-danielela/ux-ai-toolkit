### Size and scaling

The divider has a fixed thickness and stretches to the parent container width / height. 

---

## Anatomy

The divider is a horizontal (1) or vertical (2) line. 

---

## Usage

Use dividers to separate blocks on a page whenever whitespace is not enough for separation. Dividers should be used very sparingly to avoid visual clutter. 

Dividers can also be used to group actions, i.e. in a focus mode toolbar.

Some components, e.g. list items, table rows or tab rows automatically include a divider. 

### Do's and don'ts

Use dividers to group actions, i.e. in a focus mode toolbar. Avoid excessive use of dividers as this adds visual clutter. 

In table rows or list items the last item should not contain a divider.
