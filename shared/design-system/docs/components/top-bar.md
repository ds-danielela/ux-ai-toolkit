### Size and scaling

If used in combination with the main menu at the left side (for larger responsive layouts), the top bar stretches to the available remaining width to the right of the main menu. For smaller responsive layouts (S - M) it stretches to the full width of the screen.

The height of the top bar depends on its content.

---

## Anatomy

The top bar includes various icon buttons at the right (1) and an optional avatar (2) for accessing the user menu. For responsive layout M and S it includes a burger icon button at the left (3) which displays the main menu as an overlay.

---

## Usage

Use the top bar to provide access to global functionality such as navigation, system notifications, the user menu etc.

**Please note:** Corresponding layout templates already contain a top bar.

There are several types and variations of the top bar. Choose the type depending on the use case.

### Do's and don'ts

Only offer most important global functionality within the top bar. Don't overcrowd it.
