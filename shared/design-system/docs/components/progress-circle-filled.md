## Design spec

### Size and scaling

The progress circle filled has a fixed width and height depending on the responsive layout. The border has a thickness of 4 px.

---

## Anatomy

The progress circle filled contains circle (1) and an indicator (2) that either rotates on the circle or fills it to indicate how much the process has progressed. Furthermore it contains content, such as an instructive illustration (3).

---

## Usage

Use a progress circle filled to show the current progress of a running process in combination with content, such as an instructive illustration.

A progress circle filled can be determinate or indeterminate.

| Type | Usage |
| --- | --- |
| Determinate progress circle filled | Use the determinate progress circle filled if the actual progress can be calculated. This variant should be used preferably, as the user gets an idea of how long he has to wait. |
| Indeterminate progress circle filled | Use the indeterminate progress circle filled if the actual progress cannot be determined. |

### Do's and don'ts

Only use the progress circle filled if the content also provides information that is relevant or supportive to the user. Do not just use it for decorative purposes.

Do not introduce semantic colors on a progress circle filled. To indicate a successful or critical outcome of a progress, use another notification mechanism instead.
