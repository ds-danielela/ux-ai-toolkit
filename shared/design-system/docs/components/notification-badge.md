## Usage

Combine an icon with a notification badge to point the user's attention to the respective component, in case there are i.e. unread notifications in the notification menu. The notification badge can also be used to communicate status information, like case severity, at first glance.

### Do's and don'ts

Do not change the position of the notification badge. It should always be displayed at the top right.
