### Size and scaling

The avatar has a fixed size of 40px for large size and 32px for small size.

## Usage

Use avatars for a thumbnail representation of a person or entity.  

Avatars themselves are not interactive. The only exception is the avatar triggering the user menu.

**Skeleton state**

The avatar is displayed as a simple circular skeleton in the skeleton state, matching the size of the final avatar component.

![Avatar: Skeleton](https://zeroheight-uploads.s3.eu-west-1.amazonaws.com/29de5e9ea31031423dc0fe?X-Amz-Algorithm=AWS4-HMAC-SHA256&X-Amz-Credential=AKIA3AVNYHQK6SJI2PWU%2F20260508%2Feu-west-1%2Fs3%2Faws4_request&X-Amz-Date=20260508T133057Z&X-Amz-Expires=604800&X-Amz-SignedHeaders=host&X-Amz-Signature=c3d2eb284a50f7c646cc2bba44a87b5a8b8178a01c191e4275cc806f84092e81)

### Content guidelines

* Initials should always be reflected with two letters: first name and last name, i.e. LH for Linda Hall. 

### Do's and don'ts

Avoid using an avatar when it is likely that you won't have an image to show, i.e. for patients.
