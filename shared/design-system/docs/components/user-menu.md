### Size and scaling

Similar to the large avatar, the user menu button has a fixed width and height of 40 px.

The user menu has a fixed width of 328 px. The height depends on its content.

---

## Anatomy

The user menu (1) is triggered by the user menu button (2). The upper part of the user menu includes the user's avatar (3) as well as name and role (4). The main part of the user menu includes option list items with icons (5).

---

## Usage

Use the user menu to provide the user with access to user-specific settings and supplementary functions.

**Please note:** Some components and templates already include a user menu button.

If the user clicks on the user menu button, the user menu is shown below. The user menu is closed again when the user clicks on an item, on the user menu button or on the area outside the menu.

### Content guidelines

* Always use clear and concise labels for the user menu items.

### Do's and don'ts

Use sentence case for user menu items.

Use icons for all user menu items.
