## Anatomy

The password field is a specific text field (1) that contains an icon to the right (2) allowing the user to show or hide the password. It can optionally also contain an info button (3).

---

## Usage

Use a password field to allow the user to specify or enter a password.

Text within the password field is initially obscured. By clicking on the icon on the right of the password field, the text is unveiled.

### Content guidelines

* Clearly communicate the password requirements to the user by using the help text.

### Do's and don'ts

Provide useful error messages that guide users to solve the problem.
