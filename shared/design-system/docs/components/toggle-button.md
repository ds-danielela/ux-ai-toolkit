### Size and scaling

The toggle button has a fixed width and height.

---

## Anatomy

Toggle buttons contain an icon (1), a label (2) or both (3).

---

## Usage

Use a toggle button to allow the user to toggle a function on and off, i.e. measurements on a x-ray image.

If toggle buttons only consist of an icon, they should always be combined with a tooltip.

### Content guidelines

* Use icons with a metaphor that is universally understood.

### Do's and don'ts

Do not change the icon if the toggle button is in selected state. Only the background of the button changes.
