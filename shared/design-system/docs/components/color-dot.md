### Size and scaling

Color dot is used to provide visual aid to material selection, or additional cue on providing annotation.

---

## Anatomy

Color dot (1) is a filled shape used to represent color. When the dot lacks sufficient contrast against the background, a border is applied to distinguish it (2) (3). This border is automatically added by the code component based on luminosity values.

Color dot can accompany other components like tags (1), (2), and be used inline in list item's headlines (3).

---

## Usage

Use color dot to display color-related information in headers or measurements, where the color represents information visible on screen.

### Content guidelines

* Make sure to use color dot on contrasting background.

### Do's and don'ts

When the color dot you're using has a color similar to the background, enable the luminosity border toggle in your design. This more or less accurately reflects how the code component behaves.
