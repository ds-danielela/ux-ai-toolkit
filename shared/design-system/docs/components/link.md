### Size and scaling

The link will shrink to the width of the text.

## Anatomy

A link consists of its label (1). 

---

## Usage

The link can be used stand-alone or inline within a paragraph of text. 

When displayed inline, the text size of the surrounding text is applied. 

Use links to enable the user to:

* navigate to another page within the application.
* navigate to an external page.
* jump to a section on the same page.

Links can also be used for emails or phone numbers. 

### Content guidelines

* A link should not contain more than three words. 
* Link labels should be descriptive and specific. Don't just write "Click here". 

### Do's and don'ts

Write in sentence case.

If the link is used at the end of a sentence, it should not include the punctuation mark or comma. 

Use links for navigating and not for triggering actions.
