# Elevation

> Source: https://zeroheight.com/0b922d40b/v/latest/p/126c5a-elevation
> Introduction: Different elevation levels create depth and visual hierarchy in an application.

---

## Elevations

The elevation system is made by five layers, in which elements can be placed in order to provide depth and cast shadows.

---

## Usage

By applying different shadow levels or layering, elevation helps define the importance of UI elements, establish a clear structure, and guide the user's attention.

**Applying elevation in Figma**

All elevation tokens are provided as Figma styles, composed of individual Figma variables. Since the opacity varies depending on the theme, elevations also respond to the mode switch (see [Color](https://zeroheight.com/0b922d40b/p/847b05)).

### Do's and don'ts

- **Do**: Use elevation for interactive or layered elements (e.g. cards, modals, dropdowns) to convey depth.
- **Don't**: Use elevation for visually grouping or decorative purposes — since shadows are usually associated with clickable elements.
