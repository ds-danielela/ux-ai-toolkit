# Typography

> Source: https://zeroheight.com/0b922d40b/v/latest/p/78a24b-typography

The DS Design System uses the typeface **Be Vietnam Pro** which can be downloaded from the [Google Fonts page](https://fonts.google.com/specimen/Be+Vietnam+Pro?query=be+vietnam).

For languages that require special characters not covered by Be Vietnam Pro, **Noto Sans** is used as a fallback.

---

## Typography decision tokens

Typography tokens are a composition of the individual typography tokens such as font size, font style, line height and letter spacing. A range of different font styles can be used for different purposes.

**Please note**: For responsive layout S, "Small" tokens apply (see [Responsive spacings and typography](https://zeroheight.com/0b922d40b/v/0/p/24f1b2-reponsive-spacing-and-typography)).

---

## Usage

**Using typography in Figma**

All typography tokens are provided as Figma text styles, composed of individual Figma variables. Since the font size varies depending on the responsive layout, text styles respond to the mode switch (see [Responsive spacing and typography](https://dentsplysirona.zeroheight.com/styleguide/s/82440/p/24f1b2-reponsive-spacing-and-typography)).

### Do's and don'ts

- **Do**: In design and development, only combine font styles with a valid text color token.
- **Don't**: Apply arbitrary colors to text outside of the defined text color tokens.
