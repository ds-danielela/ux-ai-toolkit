## Icon sizes

The default size is 24 x 24 pixel. The current icons can be used in three different sizes. 

The used sizes are:

L - 48 pixel

M - 24 pixel 

S - 16 pixel

## Usage

For the designers, all icons are provided as components in a separate Figma library. To access the icons, the library must be included. 

![Icons: Include Library](https://zeroheight-uploads.s3.eu-west-1.amazonaws.com/bca11b4f50f0931c228d3c?X-Amz-Algorithm=AWS4-HMAC-SHA256&X-Amz-Credential=AKIA3AVNYHQK6SJI2PWU%2F20260508%2Feu-west-1%2Fs3%2Faws4_request&X-Amz-Date=20260508T134408Z&X-Amz-Expires=604800&X-Amz-SignedHeaders=host&X-Amz-Signature=74c23bf35301c2897d3ebfc3b40ebec35932871783c5c5bbebd1e99eb0490e48)

**Icons: Include Library**

---

The icons are available in the base size M of 24px. The components are created in a way that they can scale so that they can also be used in the sizes S and L. 

For optimized rendering, the icons are converted to an icon font within the development. 

### Do's and don'ts

Only use the icons in the supported sizes S, M and L. Don’t scale icons to other sizes. Stick to the icon sizes that are provided in the components.
