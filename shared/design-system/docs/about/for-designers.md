## Figma

To design with DS Design System you can have **the most recent version of** **[Figma](https://www.figma.com/downloads/)** **installed or use their** **[web application](https://www.figma.com/)**.

## Download and install font

Download typeface Be Vietnam Pro from the [Google Fonts page](https://fonts.google.com/specimen/Be+Vietnam+Pro?query=be+vietnam) and install it.

## Insert components

To insert components, drag and drop them from the assets tab.

Alternatively, you can use the "Actions" menu in the bottom toolbar to search for components.
