# Elevation

Different elevation levels create depth and visual hierarchy in an application.

---

## Elevations

The elevation system is made of five layers in which elements can be placed to provide depth and cast shadows. Both light and dark themes have corresponding shadow styles for each level.

| Elevation | Usage |
|---|---|
| `elevation5` | — |
| `elevation4` | — |
| `elevation3` | Popovers, side sheets |
| `elevation2` | Primary buttons |
| `elevation1` | Buttons, cards |
