# UX AI Toolkit

AI-powered tools for UX designers working on medical/dental software.
Built on Claude. Runs on your laptop.

---

## What this is

Two tools that sit alongside your existing Figma workflow — one for thinking, one for checking.

## Where each tool fits in your workflow

```
NEED arrives
    ↓
[Skill] Synthesise inputs → design brief
    ↓
Open Figma
    ↓
[Skill] Check screens before sync → fix blockers
    ↓
Weekly sync
    ↓
Screens on staging
    ↓
[Agent] Full audit → structured report
    ↓
Handoff
```

Your Figma work doesn't change. The tools add checkpoints before and after it.
